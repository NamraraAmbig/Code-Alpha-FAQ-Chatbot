# 🤖 FAQ Chatbot

An AI-powered FAQ chatbot built using Python, NLP, and cosine similarity.

## 📌 Project Overview

This chatbot answers user questions by matching them with the most similar FAQ using Natural Language Processing (NLP) techniques.

The chatbot:
- Accepts user questions
- Finds the most similar FAQ
- Returns the best matching answer
- Shows confidence percentage

---

## 🚀 Features

- Greeting support (Hi, Hello, Hey)
- FAQ question matching
- Confidence score display
- Typing effect
- Colored chatbot interface
- NLP-based text similarity

---

## 🛠 Technologies Used

- Python
- NLTK
- scikit-learn
- Colorama
